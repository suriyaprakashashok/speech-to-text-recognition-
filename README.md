# 🎙️ AI Speech-to-Text Web Application

A modern Speech-to-Text web application built using Flask and Python. The application allows users to upload WAV audio files and converts spoken words into text using Google's Speech Recognition API.

## Features

- 🎤 Speech-to-Text Conversion
- 📂 WAV File Upload
- ⚡ Fast AI-based Recognition
- 💎 Modern Responsive UI
- 🐍 Built with Flask

## Technologies Used

- Python
- Flask
- SpeechRecognition
- HTML5
- CSS3

## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Speech-To-Text-AI.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
python app.py
```

Open your browser

```
http://127.0.0.1:5000
```
